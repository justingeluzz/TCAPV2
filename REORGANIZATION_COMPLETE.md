## 🚀 How to Start TCAP v2

### Option 1: Use Startup Scripts (Recommended)
```powershell
# Windows Batch
scripts\start.bat

# PowerShell  
scripts\start.ps1
```

### Option 2: Manual Start
```powershell
# 1. Navigate to backend folder
cd backend

# 2. Start the backend server
python backend.py

# 3. Open frontend/index.html in your browser
```
